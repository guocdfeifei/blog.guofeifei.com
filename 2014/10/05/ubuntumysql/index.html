ubuntu上mysql的安装以及基本用法

By guoff

2014-10-05 更新日期:2015-06-24

文章目录
  1. 1. bind-address = 127.0.0.1

1 安装

sudo apt-get install mysql-server

2 使用

3 # 登录
mysql -uroot -p

4.mysql服务的启动与停止

安装完后mysql服务默认是启动状态。

停止服务

sudo /etc/init.d/mysql stop

启动服务

sudo /etc/init.d/mysql start

重启mysql

sudo /etc/init.d/mysql restart

5.让外网可以访问mysql
改mysql配置文件
sudo vi /etc/mysql/my.cnf
找到该行并注释掉

bind-address = 127.0.0.1

保存修改后的my.cnf文件（ctrl+zz）
重启mysql
sudo /etc/init.d/mysql restart

6.分配mysql用户

mysql -uroot -p
mysql>grant all privileges on 数据库名.* to ‘账号’@’%’ identified by ‘口令’;
mysql>flush privileges;
mysql>exit

7.对于django或其他框架，mysql默认设置不支持中文，需要设置后再建表方可。

具体修改默认设置方法如下：

http://blog.csdn.net/moxiaomomo/article/details/8539578

主旨为：查，停，改，起，建，查

show variable like ‘%character%’

service mysql stop

vim /etc/mysql/my.cnf

打开my.cnf后，在文件内的[mysqld]下增加如下两行设置:

character_set_server=utf8

init_connect=’SET NAMES utf8’

service mysql start

重新建表

再去查看

mysqlubuntu
