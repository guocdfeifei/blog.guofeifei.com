I’ve heard that ubuntu is user friendly.

常用命令：

重启：reboot

查找从当前目录下起，含有某字符串的文件

find .|xargs grep -ri “get_calendar_entries”

常用快捷键：

进入终端界面：ctrl+alt+F1

返回桌面界面：ctrl+alt+F7

启动顺序更新

sudo update-grub

编辑启动配置

sudo chmod +w /boot/grub/grub.cfg

sudo gedit /boot/grub/grub.cfg

重点属性

set default=0 //默认启动项

set timeout=10 //启动时选择等待时间

给文件及子目录赋权

sudo chmod 777 1260OT_Code -R

在命令行里,输入sudo nautilus
之后输入你的用户的密码,会弹出一个目录窗口来,可以复制到这里来的,在这里也可以删掉root的文件.
